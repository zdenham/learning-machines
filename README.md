# Learning Machines

A daily math practice curriculum building toward deep understanding of neural networks, LLMs, and artificial intelligence.

## How it works

The commitment is simple: do math every day. Work through problems, proofs, and derivations that build toward understanding modern AI from the ground up.

Progress doesn't have to be linear — some days you revisit a concept, go on a tangent, or spend extra time on something that clicks. The course outline is a guide, not a rail. What matters is showing up daily and doing the work.

Topics take as long as they take. Don't estimate time, don't rush to the next section. Move on when the material feels solid, not when a schedule says to.

- **`course/`** — The curriculum outline and topic sequence.
- **`problem-sets/`** — Full problem statements for each topic. Numbered sequentially (e.g. `01-vectors-dot-products.md`).
- **`progress/`** — Daily logs tracking what was covered and what's left. One file per day (`YYYY-MM-DD.md`), linking to the relevant problem set.

## Getting started

Check `course/outline.md` for the full curriculum. Look at `progress/` to see what's been completed, then pick up wherever makes sense.
